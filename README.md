## 🧪 Test Case Details

* Total Test Cases: 50
* Type: Negative Test Cases
* Input Length Categories:

  * S (Short)
  * M (Medium)
  * L (Long)

Each test case includes:

* Singlish Input
* Expected Sinhala Output
* Input Type Classification
* Rationale / Evidence

The following Singlish input types were tested:

* Question Forms
* Command Forms
* Greetings
* Requests & Responses
* Repeated Words
* Punctuation Usage
* Spelling Variations
* English Word Insertions
* Digital / Technical Terms
* App & Platform Names
* Numbers & Currency
* Time & Date Formats
* Measurements
* Slang & Casual Language
* Online Identifiers (URLs, Emails)
* Emojis
